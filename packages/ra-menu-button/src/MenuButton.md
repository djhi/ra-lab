## `<MenuButton>`

### Props

`<MenuButton>` expects the following props:

| Name           | Required | Type           | Default                   | Description |
| -------------- | -------- | -------------- | ------------------------- | --- |
| `children`     | Required | `string`       | -                         | The menu items |
| `button`       |          | `string`       | `<MenuButton.IconButton>` | The button that opens the menu on click |
